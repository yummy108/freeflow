# 🚀 FreeFlow — Quickstart (5 Minute Version)

Agar BEGINNER_GUIDE.md bahut lamba lage, to bas ye 5 steps follow karo:

## Step 1: GitHub Account
→ https://github.com/signup (free, 2 min)

## Step 2: Gemini API Key
→ https://aistudio.google.com → "Get API Key" → Copy karo

## Step 3: Fork Repo
→ (URL dunga jab live ho) → "Fork" button → "Create fork"

## Step 4: Add Secret
→ Tumhare forked repo → Settings → Secrets and variables → Actions → New secret
→ Name: `GEMINI_API_KEY`
→ Value: Wo key jo Step 2 me copy ki

## Step 5: Enable Pages
→ Settings → Pages → Source: main branch / root → Save

## ✅ Done!
→ Website: `https://TUMHARA-USERNAME.github.io/freeflow`
→ Pehli video: Drop a `.md` file → Generate
