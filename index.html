🛠️ How to set this up (5 min)

    
    
      
  1. Fork this repo on GitHub (must be public for free Actions).
    2. 
      
  2. Get free API keys (all optional but recommended):
        
      
          
    • Gemini API — 1,500 req/day free (research + script)
      • 
          
    • Groq — 1,000 req/day free (super fast LLM)
      • 
          
    • Pexels — Free stock video API
      • 
        
    
        Note: Edge TTS, Pollinations, Pixabay need NO key.
      
    3. 
      
  3. Add secrets: Settings → Secrets and variables → Actions → New repository secret
        
      
          
    • GEMINI_API_KEY
      • 
          
    • GROQ_API_KEY
      • 
          
    • PEXELS_API_KEY
      • 
        
    
      
    4. 
      
  4. Enable GitHub Pages: Settings → Pages → Source: main branch / root. Your site will be live at https://<user>.github.io/<repo>.
    5. 
      
  5. Add topics in Mode 2 above, then commit pipeline/topics.json via the web UI or manually.
    6. 
      
  6. For hourly auto-run: GitHub Actions workflow is already set to run every hour. Just commit your topics.json.
    7.
